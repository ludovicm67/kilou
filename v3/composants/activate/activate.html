KILOU doit être activé !

Et oui, souvent, des personnes copient et modifient des fichiers de certains programmes. C'est donc pour cela que nous avons mis en place ce système d'activation ! Cette activation est totalement gratuite mais vous devez être patient car il faut 2-3 jours en moyenne pour que KILOU soit activé. Vous devrez remplir un formulaire correctement pour que nous puissions vous donner une clé d'activation. Dans votre session KILOU, quand le message "Votre version de KILOU doit être activé" apparaît, en bas de la fenêtre, il y a deux possibilités : 

 - soit : Activer KILOU.

 - soit : Entrer la clé d'activation. 

Pour venir sur cette page, vous avez cliqué sur 'Activer KILOU'. Une fois que vous recevrez la clé par e-mail, vous devrez aller à cette fenêtre et cliquer sur 'Entrer la clé d'activation'. une nouvelle page s'ouvrira alors et vous devrez entrer la clé d'activation. Si vous avez rentré une mauvaise clé, vous retournerez à la fenêtre précédente ( Votre version de KILOU doit être activé ). Mais si vous avez rentré la bonne clé, vous accèderez à un site Web ( mais pour cela, veuillez vérifier que votre ordinateur est bien connecté à INTERNET ) . Sur cette page Web, un paquet d'installation vous sera proposé. Acceptez ! Si vous refusez, votre version de KILOU ne pourra pas être activé correctement ! Une fois que vous avez installé ce paquet d'installation final, ( en .zip ) ouvrez-le. Ouvrez le fichier 'Lisez-moi.txt' ( Ce fichier vous expliquera la suite et où et comment installer les nouveaux fichiers ! ) . Voilà, cliquez sur le bouton ci-dessous pour aller remplir le formulaire d'activation pour KILOU :
